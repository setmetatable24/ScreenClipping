# ScreenClipping
This program is a Windows 10 equivalent of ScreenClippingHost.exe. I had a problem where my system crashed after taking a screenshot with ScreenClippingHost.exe, so I decided to create this program. If anyone has a similar problem, you can install my program, ScreenClipping, which will help you with this! Here you can copy an image (screenshot) you've taken, save it as PNG or JPEG, draw on photos with different palettes, achieve pixel quality, and set high contrast and brightness. The program may have some shortcomings (a couple of bugs), but I will fix them as the application develops. If anyone is interested, the program is written in pure C, without any GUI libraries from the internet. So if anyone needs it, use it!
